# Devin Tjaden Personal Website

A clean, dark-first personal website with both a portfolio and a blog, built with plain HTML/CSS/JS.

## Pages

- `index.html` - home page (portfolio-first overview)
- `portfolio.html` - full portfolio page
- `resume.html` - resume and full work history
- `blog.html` - blog index
- `blog-post.html` - sample blog article page

## Core Files

- `styles.css` - layout, theme, and visual design
- `script.js` - light/dark mode toggle and dynamic year

## Customize Content

Update text in:

- `index.html` for hero, featured projects, and contact
- `portfolio.html` for case studies and project descriptions
- `blog.html` for post listings
- `blog-post.html` for article content

## Run Locally

Open `index.html` in your browser, or run a local server from this folder.

## Publish

You can deploy this quickly with:

- GitHub Pages
- Netlify
- Vercel
